About CSS

Embedding CSS in HTML

  • Inline - By using the style attribute
  • Embeded - By using the style element
  • External - By using an external file and the link element

CSS Selectors

A little about CSS selector types
  • Select by name (type) - Stating the name of the element
  • Select by class - Stating the name of the class with a . before it
  • Select by id - Stating the name of the id with # before it DO NOT USE IT!
  • Select by attribute - Stating the name of the attribute between [] (possibly also with the attribute value)DO NOT USE IT!
  • Select everything - Using * selector

CSS Combinators

  • Ancestor - By placing space between the selector
  • Direct Child - By using the > between the selectors
  • And - By placing the two selectors without space between them
  • Direct Sibling - By using the + between the selectors
  • General Sibling - By using the ~ between the selectors
